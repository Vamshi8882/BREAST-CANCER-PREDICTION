# BREAST-CANCER-PREDICTION
A machine learning–based system for early breast cancer detection using the Breast Cancer Wisconsin dataset. Data preprocessing, feature scaling, and train–test split are applied. Logistic Regression and Decision Tree classify tumors as benign or malignant and are evaluated using accuracy, precision, recall, F1-score, and confusion matrix.


**ALGORITHM:**
Step 0: Start
Step 1: Import Required Libraries
Import Python libraries such as NumPy, Pandas, Matplotlib, Seaborn, and Scikit-learn for data processing, visualization, and model building.
Step 2: Load Dataset 
Load the Breast Cancer Wisconsin dataset from a CSV file into a Pandas DataFrame.
Step 3: Data Preprocessing
Remove unnecessary columns, handle missing values, and encode the target variable (Malignant = 1, Benign = 0).
Step 4: Feature Selection
Separate the dataset into input features (X) and target variable (y).
Step 5: Split Dataset
Divide the dataset into training and testing sets using an 80:20 ratio.
Step 6: Feature Scaling
Apply standardization to the training and testing data for Logistic Regression to normalize feature values.
Step 7: Train Logistic Regression Model
Train the Logistic Regression classifier using the scaled training data.
Step 8: Train Decision Tree Model
Train the Decision Tree classifier using the training dataset.
Step 9: Make Predictions
Predict the class labels (malignant or benign) using both trained models on test data.
Step 10: Model Evaluation
Evaluate both models using accuracy, confusion matrix, precision, recall, and F1-score.
Step 11: Compare Model Performance
Compare the accuracy of Logistic Regression and Decision Tree to identify the better-performing model.
Step 12: Display Results
Visualize confusion matrices and display evaluation metrics.
Step 13: End
